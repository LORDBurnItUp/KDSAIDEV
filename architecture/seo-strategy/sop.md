# SEO Strategy SOP (A.N.T. Layer 1)

## Mode 1: Article/Page Optimization
1. **Intake:** Ask the 7 required Discovery questions (Keywords, Audience, Intent, Geography, LSI, Competitors, Constraints).
2. **Research:** 
    - Search target keyword.
    - Fetch top 3-5 competitors.
    - Extract Title/H1, Headings, Word Count, and LSI keywords.
3. **Gap Analysis:** Compare user content vs. competitors (Content, Structure, Depth, Keywords).
4. **Optimization:** Rewrite article matching search intent and optimizing for E-E-A-T and LSI density.
5. **Output:** Generate `seo-report-[slug].html`.

## Mode 2: Full Website Audit
1. **Intake:** Ask the 7 site-wide questions (Target KWs, Business Type, Geography, Competitors, Goals, LSI Priorities, Known Issues).
2. **Crawl:** Map up to 15 key pages (Homepage, Nav, Pillar content).
3. **Analyze:** Check Technical (HTTPS, Robots, Sitemap), Header/Meta, and Content metrics per page.
4. **Cross-Page:** Detect duplicates, cannibalization, and internal link health.
5. **Scoring:** Compute Technical (25%), Content (25%), Architecture (20%), Cross-Page (15%), and Social (15%) scores.
6. **Output:** Generate `seo-strategy-[domain].html`.

## Deterministic Rules
- Always use high-contrast dark theme for HTML reports.
- LSI extraction must involve 20-40 terms.
- Never keyword stuff; prioritize semantic relevance.
